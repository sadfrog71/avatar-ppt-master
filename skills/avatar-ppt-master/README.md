# Avatar PPT Master

Avatar PPT Master 是面向内部同事使用的本地 PPT 生成 Skill。它先分析汇报目标、受众、结论和页面任务，再从预置主题中组合页面，生成可预览、可编辑和可导出的演示文稿。

当前版本：`0.2.0`

## 适用场景

- 内部项目汇报、管理层提案和阶段复盘；
- 行业研究、竞品分析、方案说明和培训材料；
- 水务运营、环境管理、数字化建设和 AI 落地方案；
- 需要继续编辑、归档或导出的正式演示材料。

## 输出内容

- 本地 HTML 演示，可横向翻页并在浏览器中编辑；
- PDF 文件，用于评审和归档；
- 可编辑 PPTX，保留文字与主要图形对象；
- `composition-plan.json`，记录页面任务、结论标题和视觉表达计划；
- 本次演示使用的本地素材副本，不依赖原始素材路径。

## 使用方式

安装后，在 Codex、Claude Code 或兼容 Agent 中指定 `avatar-ppt-master`，并说明：

- 汇报目标和受众；
- 期望页数；
- 重点结论、数据、风险和行动项；
- 指定主题或允许 Skill 推荐；
- 最终输出格式。

示例：

```text
使用 avatar-ppt-master，把这份调研材料整理成 12 页管理层汇报。
使用 theme13 水务环境风，重点表达经营现状、主要风险和下一步行动，输出可编辑 PPTX。
```

## 内容生成特点

- 先规划叙事结构，再选择页面组件；
- 页面标题优先写成结论，不只写章节名称；
- 每页只承担一个主要信息角色；
- 图表、矩阵、路线图和风险页必须给出读图结论；
- 视觉结构服务于内容，不为填满页面而堆叠模块；
- 生成后检查无关默认文案、素材路径和导出完整性。

## 安装

将整个 `avatar-ppt-master` 目录复制到本机 Skill 目录，并重新打开会话。

Codex 默认目录：

```text
~/.codex/skills/avatar-ppt-master
```

Claude Code 默认目录：

```text
~/.claude/skills/avatar-ppt-master
```

本机需要 Node.js 20+ 和 npm。首次生成时会在 Skill 自带的 `project/` 目录准备依赖。

## 内部数据说明

生成器支持本地素材暂存与离线成品，但所接入的 AI Agent 或模型可能有独立的数据处理策略。处理企业内部材料时，仍需遵循组织的数据分类、脱敏和外发要求。
